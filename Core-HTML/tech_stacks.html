Full Stack Web Development

List of tech stacks:

FrontEnd:

HTML:
HTML stands for Hypertext Markup Language, It is used to define the structure of web page.
CSS:
CSS stands for Cascading Style Sheet, It basically make the web page beautiful and colorful.
Javascript:
Javascript basically used to make the web page interactive and for client site validation as well.

BackEnd:

A Server Side language:
The server side programming language is used to interact with the server and server side validations.
DBMS:
DBMS stands for Database Managment System, It is used to store the data that is required for the web applications.
