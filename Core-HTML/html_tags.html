Some of the HTML Tags:

  1. html :The root element that contains all other HTML elements.
    2. head tag: Contains meta-information about the document, such as the title and links to external resources.
  2. meta tag: Provides metadata about the document, such as character encoding and author information.
  3. title tag: Sets the title of the web page, which appears in the browser's title bar or tab.
  4. link tag: Links to external resources, such as stylesheets or icons
  5. style tag: Contains embedded CSS (Cascading Style Sheets) for styling the document.
  6. body tag: Contains the main content of the web page, including text, images, and other media.
  7. heading tags: Headings, used to structure and format text into different levels of headings, with h1 being the highest level.
  8. p tag: Defines a paragraph of text.
  9. a tag:Creates hyperlinks to other web pages or resources.
  10. img tag:Embeds images in the document.
  11. ul tag:Defines an unordered list (bulleted list).
  12. ol tag: Defines an ordered list (numbered list).
  13. li tag:Represents list items within ul or ol.
  14. table tag:Defines a table for displaying tabular data.
  15. hr tag:Inserts a horizontal rule (line) to separate content.
  16. br tag:Inserts a line break within text.
